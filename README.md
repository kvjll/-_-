## Мой JavaProject 
Менеджер студентов - это десктопное приложение для учета успеваемости студентов, разработанное на Java с использованием JavaFX. Программа предназначена для преподавателей и учебных заведений, позволяя удобно вести учет оценок студентов и рассчитывать средний балл.

## Основные функции:
- Добавление новых студентов в базу
- Редактирование оценок за задания
- Удаление студентов из системы
- Автоматический расчет среднего балла
- Табличное представление данных с возможностью редактирования
- Валидация вводимых данных

## Установка и запуск
Предварительные требования

- Java 8 или выше
- JavaFX 25.0.1
- Операционная система: Windows
## Шаги по запуску: 
1) Скачайте проект:
     ```git clone ```
     ```cd untitled3```
3) Компиляция проекта:
     ```javac -d out/src src/*.java```
4) Запуск приложения:
     ```java -cp out EchoApp```

## Пример использования
Добавление студента: 
1) Заполнить все поля:
   - Имя
   - Фамилия
   - Группа
   - Оценка за 1 задание
   - Оценка за 2 задание
2) Нажмите кнопку "Добавить"

Редактирование студента: 
1) Выберите студента в таблице
2) НАжмите на кнопку "редактировать студента"
3) Измените нудны поля в форме
4) Нажмите "Добавить" для сохранения изменений
